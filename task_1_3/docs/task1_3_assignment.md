# Задача 1.3 — Витрина 101 формы

## Постановка задачи

На основе рассчитанных витрин оборотов и остатков (задача 1.2) необходимо рассчитать 101 форму за январь 2018 года.

## Витрина DM.DM_F101_ROUND_F

Содержит остатки и обороты за отчётный период, сгруппированные по балансовым счетам второго порядка (первые 5 символов номера счёта).

**Процедура:** `dm.fill_f101_round_f(i_OnDate DATE)`

Входной параметр `i_OnDate` — первый день месяца, следующего за отчётным.
Для расчёта за январь 2018 → передать `'2018-02-01'`.

## Логика расчёта периода

| Переменная | Значение |
|-----------|----------|
| from_date | i_OnDate − 1 месяц (01.01.2018) |
| to_date | i_OnDate − 1 день (31.01.2018) |
| Входящий остаток | день перед from_date (31.12.2017) |

## Поля витрины

| Поле | Источник |
|------|----------|
| FROM_DATE / TO_DATE | Начало и конец отчётного периода |
| CHAPTER | DS.MD_LEDGER_ACCOUNT_S.chapter |
| LEDGER_ACCOUNT | Первые 5 символов DS.MD_ACCOUNT_D.account_number |
| CHARACTERISTIC | DS.MD_ACCOUNT_D.char_type |
| BALANCE_IN_RUB | Остаток на 31.12.2017, рублёвые счета (currency_code = 810 или 643) |
| BALANCE_IN_VAL | Остаток на 31.12.2017, валютные счета |
| BALANCE_IN_TOTAL | Остаток на 31.12.2017, все счета |
| TURN_DEB_RUB | Дебетовые обороты за январь, рублёвые счета |
| TURN_DEB_VAL | Дебетовые обороты за январь, валютные счета |
| TURN_DEB_TOTAL | Дебетовые обороты за январь, все счета |
| TURN_CRE_RUB | Кредитовые обороты за январь, рублёвые счета |
| TURN_CRE_VAL | Кредитовые обороты за январь, валютные счета |
| TURN_CRE_TOTAL | Кредитовые обороты за январь, все счета |
| BALANCE_OUT_RUB | Остаток на 31.01.2018, рублёвые счета |
| BALANCE_OUT_VAL | Остаток на 31.01.2018, валютные счета |
| BALANCE_OUT_TOTAL | Остаток на 31.01.2018, все счета |
