# Задача 1.4 — Экспорт и импорт данных витрины 101 формы

## Описание

Выгрузить данные из `dm.dm_f101_round_f` в CSV-файл и загрузить обратно в `dm.dm_f101_round_f_v2`.

## Требования

- Скрипт экспорта: `export_f101.py`
- Скрипт импорта: `import_f101.py`
- Первая строка CSV = названия колонок
- Простое логирование в обоих скриптах
- Перед импортом: вручную изменить 2-3 значения в CSV
